# EarningsEdge
An Intelligent Earnings Call Analysis & Stock Movement Prediction System

## Features
- 🔍 EDA — explore 13,377 earnings call transcripts (2019–2023)
- 🤖 ML — predict post-earnings stock movement with LightGBM
- 💬 NLP — FinBERT sentiment analysis
- 🧠 Chat — RAG chatbot powered by LLaMA 3.3 via Groq
